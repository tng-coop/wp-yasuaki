A minimal Gutenberg block that displays "hello".

== Description ==
Adds a single dynamic block that renders the text "hello" on both front end and in the editor. No build tools required.

== Installation ==
1. Upload the `fullshow` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the **Fullshow Hello** block to any post or page.
